# Bdaf lab5

## Usage
Install dependencies:
```
$ npm i
```

Compile smart contract:
```
$ npx hardhat compile
```

Test script in ```test/Safe.test.js``` & retrieve gas-report
```
$ npx hardhat test
```

Get test coverage:
```
$ npx hardhat coverage
```
